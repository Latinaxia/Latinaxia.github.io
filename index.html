草台班子建站历程（基于hexo框架的blog搭建的个人blog）

hexo的官方文档实在是过于构式，写下本文记录简单高效的流程

前置准备

  • Git
  • Node.js
  • npm & pnpm

框架搭建

  1. 使用npm全局安装hexo
    npm install -g hexo-cli
  2. 创建一个空文件夹，进入文件夹，构建hexo
    hexo init
  3. 接下来先选择主题，hexo官网有四百多个主题供选择，选中心仪的主题后在theme文件夹中拉取
    例如我的主题是ParticleX：
    git clone https://github.com/theme-particlex/hexo-theme-particlex.git particlex --depth=1
    之后在根目录_config.yml修改
    theme: ParticleX
    以上步骤只是一个通用方式，具体配置还需参考对应主题的文档

Github-pages部署

  1. 首先在GitHub创建一个public的空仓库，确保仓库只有一条分支即main，且仓库必须为.github.io,仓库必须为空！
  2. 打开Git Bash，获取SSH密钥
    ssh-keygen -t ed25519 -C <email>
    eval "$(ssh-agent -s)"
    ssh-add ~/.ssh/id_ed25519
    由于本人是Windows系统，接下来打开powershell：
    Get-Content $env:USERPROFILE\.ssh\id_ed25519.pub | Set-Clipboard
  3. 接下来将得到的SSH密钥添加到Github中即可，可以使用以下命令查看是否添加成功：
    ssh -T git@github.com
  4. 在yml文件中添加以下配置：
    url: https://<user>.github.io/
    deploy:
    type: git
    repo: git@github.com:<user>/<user>.github.io.git
    branch: main
    message:
  5. 安装部署所需的插件：
    pnpm add hexo-deployer-git
  6. 在根目录下运行以下命令部署：
    hexo deploy
  7. 等待部署完成，打开浏览器访问https://.github.io/即可看到自己的博客啦！

草台班子建站记

大概是在八月初的时候，在狠狠视奸了Dawson和周周的个人blog之后，我也萌生了建立个人blog的想法
但是由于我执行力实在是太差，导致目前对于前端依旧一窍不通，又脸皮薄不好意思麻烦别人，这个计划就搁置了很久，直到2024-8-18:16:47，终于破土动工了

感谢ct，感谢Dawson，感谢涛涛，在这两天内不厌其烦的回答我的种种愚蠢的问题（有的东西想起来都让我汗颜），终于让我这个连vsc都用不明白的前端小白成功搭建了自己的blog并在GitHub上部署成功

接下来进入正题

在正式开始工作（老实来说我对这个词很不感冒，但这两天的遭遇让我感觉自己就是在工作）之后，我才发现建立一个个人blog比自己想象的简单一点点。

  • 不需要服务器，github-pages提供了现成的部署方式
  • 不需要自己写代码，hexo（我选用的blog框架）几乎做了所有的代码工作，你要做的只是在合适的位置加上属于自己的东西
  • 对于懒狗来说连主题都是全自动的，在我建站时hexo提供了一共408个主题供你选择
  • npm和pnpm真的超级好用，什么时候maven也能这样。。。
  • 但话说回来vsc连接GitHub总是超时，无奈只能转战IDEA来完成建站

当然，在我自己动手实操时，难免有跌跌碰碰
但是历经三天时间，在Dawson的指导下，终于成功建站并发布了自己的第一篇blog–《草台班子建站记》，如今我正坐在家中用vsc编辑着md文档（这玩意儿写md真的舒服），纠结要不要按时间对blog进行分包。

最后

再次感谢Dawson，这两天我们的聊天高达数百条，大多都是我的询问，期间他也远程操控了我的电脑。看到他最近又在开发自己的通用模板和魔改自己的blog，不禁心里有些佩服和羡慕，个人觉得能做出一些看得见的实践获得的成就感远远大于在后端摆弄无聊的逻辑与数据

  • 这里是Dawson的blog：https://www.betula.space/

在19号晚上又听闻玉玉通过了字节四道面试拿到前端实习，又狠狠的羡慕了，心潮澎湃，恭喜玉玉（以后再要他的blog）。

建站遇到的种种bug阻碍我本打算细细列出，顺便痛骂一下hexo的构式文档误人子弟，但实在太懒，今天就先到此为止吧
